# QST-079 Report: Guild Quest Matching

## Summary

Added a lightweight Guild Quest Matching foundation. Matching ranks related public/Guild Quests using category and title terms while filtering private Quest candidates so private content is not exposed.

## Changed Files

- apps/mobile/lib/features/guild/guild_quest_matching_service.dart
- apps/mobile/lib/features/guild/guild_screen.dart

- docs/qst/BACKLOG.yaml

## Implementation Notes

- Added `GuildQuestMatchingService` and `GuildQuestMatch`.
- Ranking uses local deterministic category and keyword overlap.
- Private Quest candidates are excluded.
- Guild screen now shows matching candidates with title, category, visibility, score, and reason.
- No Supabase or paid feature dependency was introduced.

## Acceptance Check

- Matching can rank related Quests by category and keywords.
- Private Quest content is not exposed.
- Guild surface shows useful candidate reasons.
- MVP Guild question and Trail reflection flows remain available.
