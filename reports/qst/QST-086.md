# QST-086 Quest Template Library

Status: Done
Date: 2026-06-21

## Goal

Help users start faster by choosing an editable Quest template for common life areas.

## Implementation

- Added Quest template model with editable Quest fields, suggested Milestones, and suggested Missions.
- Added a curated local template library for travel, health, learning, family, work, and challenge.
- Exposed the template library through Quest providers.
- Updated Quest creation to show template chips before the form fields.
- Applying a template fills Quest title, description, category, and difficulty while keeping the form editable.
- Added a preview of suggested Milestones and Missions before saving.
- Preserved the existing Quest save and Arc Guide generation flow.
