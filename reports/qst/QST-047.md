# QST-047 Arc Expression Engine

## Status

Done

## Summary

Added a centralized Arc Expression Engine that resolves Arc emotion and PNG
asset path from Quest, Mission, Trail, Reflection, celebration, concern, empty
journey, and Bond-growth context.

## Implementation Notes

- Added `ArcExpressionContext`, `ArcExpressionMoment`, and
  `ArcExpressionDecision`.
- Added `ArcExpressionEngine.resolve` for explicit context decisions.
- Added `ArcExpressionEngine.resolveJourney` for current Quest/Mission/Trail
  state.
- Connected Home and Arc Chat to use the engine for their main Arc expression.
- Kept `ArcAssetPaths` as the single source for image paths.

## Acceptance

- Arc expression selection is centralized.
- Quest, Mission, Trail, Reflection, and Bond inputs can resolve an expression.
- Existing Arc UI consumes the resolver for major Arc surfaces.
- Arc remains framed as a navigator/companion.
- No `Story` product language was introduced.
