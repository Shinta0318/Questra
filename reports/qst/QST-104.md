# QST-104 Global Scroll Behavior

## Status

Done

## Summary

Standardized Questra scroll physics and pointer support, and added safe
Pull-to-refresh to repository-backed lists.

## Implementation

- Added `QuestraScrollBehavior` at the app boundary.
- Enabled touch, stylus, mouse, and trackpad dragging.
- Uses bouncing physics on iOS/macOS and clamping physics elsewhere.
- Keeps lists scrollable when content is short so refresh gestures remain
  available.
- Added optional `RefreshIndicator` support to
  `QuestraResponsiveListView`.
- Connected refresh to Quest, Mission, and Trail repository load methods.
- Refresh is disabled for guest state because there is no remote owner to load.
- Guild derived content and Arc conversation history keep consistent scrolling
  without presenting a misleading refresh action.
