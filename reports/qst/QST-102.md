# QST-102 Responsive Layout System

## Status

Done

## Summary

Added shared compact, medium, and expanded width rules and a centered,
responsive list container for Questra's main Flutter surfaces.

## Implementation

- Added `QuestraWindowClass`, `QuestraBreakpoints`, and `QuestraLayoutSpec`.
- Added `QuestraResponsiveListView` with adaptive horizontal padding.
- Capped expanded content at 960 logical pixels by default.
- Applied narrower caps to Profile and onboarding/form flows.
- Applied the shared container to Home, Quest list/detail/create, Mission,
  Trail, Guild, Arc Chat history, Profile, and onboarding.
- Added breakpoint and expanded-width widget tests.
