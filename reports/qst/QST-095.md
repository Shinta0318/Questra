# QST-095 Mature Arc Memory Retrieval

Source QST: `DEV-QST/QST-046`

## Status

Done

## Scope

Added deterministic Arc Memory retrieval for Arc Chat context using message relevance, active Quest relationship, importance, and recency.

## Changes

- Added `ArcMemoryRetrievalService` with bounded, deterministic ranking.
- Boosted memories that match the current message or belong to an active Quest.
- Preserved importance and recency as retrieval signals.
- Excluded sensitive memories from Arc Chat context by default.
- Connected retrieval to the Arc Chat send path.
- Added targeted ranking, sensitivity, and limit tests.
