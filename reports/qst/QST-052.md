Added a shared Arc-guided empty state component and applied it to the MVP surfaces where an empty journey can otherwise feel like a dead end.

## Implementation Notes

- Added `ArcEmptyState` with Arc expression, Japanese guidance, and a single clear CTA.
- Quest empty state now points to Quest creation.
- Mission empty state now points back to Quest, where Mission generation begins.
- Trail empty state now points back to Quest and the Quest to Mission to Trail loop.
- Guild empty state now asks the user to prepare a Quest or Trail before bringing context to Guild.
- Arc Chat now surfaces an Arc Memory empty state that explains how journey context accumulates.

## Acceptance Check

- Quest, Mission, Trail, Guild, and Arc Memory empty states use Arc guidance.
- Empty-state copy points to one clear next action.
- Expressions use the empty journey emotional tone through the Arc expression system where relevant.
- User-facing copy avoids legacy terminology and does not frame Arc with prohibited language.
