# QST-060 Report: Arc Daily Greeting

## Summary

Added a deterministic Arc Daily Greeting layer for Home. Arc now greets the user with Japanese copy that adapts to recent Trail activity, open Mission state, active Quest progress, and first-run/quiet-day fallback states.

## Implementation Notes

- Added `ArcDailyGreetingService` and `ArcDailyGreeting`.
- Home now reads current Quest, Mission, Trail, and Profile state through Riverpod and passes it to the greeting service.
- Greeting priority is:
  1. Recent Trail within one day.
  2. Open Mission, with a gentler concern variant when the Mission is stale.
  3. Active Quest progress.
  4. First Quest prompt or quiet-day Trail prompt.
- Arc emotion changes with greeting context and keeps Arc framed as a navigator/companion.
