# QST-088 Onboarding Personalization

Status: Done
Date: 2026-06-22

## Goal

Personalize the first-run experience with Arc naming, Quest tendency, and Signal frequency while keeping setup lightweight.

## Implementation

- Added profile preference fields for Arc name, Quest interest, and Signal frequency.
- Extended onboarding to collect captain name, Arc name, Quest interest, Signal frequency, and first Quest.
- Updated onboarding Quest creation so the first Quest category reflects the selected interest.
- Updated Arc Daily Greeting to reflect Arc name and Quest interest in the first-route message.
- Updated Mission Signal generation to respect quiet, balanced, and frequent Signal frequencies.
- Added Supabase migration for `arc_name`, `quest_interest`, and `signal_frequency` profile columns.
- Kept local fallback behavior if remote profile preference persistence is unavailable.
