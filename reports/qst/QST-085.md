# QST-085 Dream Board V1

Status: Done
Date: 2026-06-21

## Goal

Let users collect Quest-scoped visual inspiration and reference notes without disrupting the existing Quest -> Mission -> Trail loop.

## Implementation

- Added Dream Board item model with item types for vision, reference, tools, Guild hints, and future generated backgrounds.
- Added an in-memory Dream Board repository boundary for MVP/Beta stability.
- Added a Riverpod Dream Board controller with optimistic local add/remove behavior.
- Rebuilt the Quest Detail Dream Board section from static chips into an interactive Quest-scoped surface.
- Added Arc-guided empty state copy and quick actions for adding a vision or Star Map reference.
- Kept Trail media upload/delete/replace behavior unchanged and avoided new backend/media complexity.
