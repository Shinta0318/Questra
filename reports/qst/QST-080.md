# QST-080 Report: Guild Safe Posting Review

## Summary

Added a lightweight Arc Review for Guild question drafts. The review flags contact information, location-sensitive language, and pressure wording before users copy or post a Guild consultation.

- apps/mobile/lib/features/guild/guild_safe_posting_review_service.dart

## Implementation Notes

- Added deterministic `GuildSafePostingReviewService`.
- Direct contact information is treated as blocked.
- Location and pressure wording are treated as caution issues.
- Guild question card now shows an Arc Review panel.
- No backend moderation dependency or platform integration was added.

## Acceptance Check

- Review flags obvious personal contact information.
- Review warns on location-sensitive and pressure wording.
- Normal Quest consultation text passes.
- User can revise copy before posting.
- Arc is presented as a navigator/reviewer, not an assistant.
