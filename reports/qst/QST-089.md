# QST-089 Beta Analytics Events

Status: Done
Date: 2026-06-23

## Goal

Define and implement privacy-conscious beta analytics boundaries for core app behavior.

## Implementation

- Added stable beta analytics event names for Quest, Mission, Trail, Arc Chat, Guild draft, media, and onboarding actions.
- Added client-side payload allowlist and blocked-key sanitizer.
- Added local-safe analytics repository and centralized analytics service.
- Emitted safe analytics events from Quest creation, Mission completion, Trail posting, media attachment, Arc Chat send, Guild draft copy, and onboarding completion.
- Documented event taxonomy, allowed payload, blocked payload, and future provider requirements.
