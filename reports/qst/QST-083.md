# QST-083 Report: Arc 3D Readiness Architecture

## Summary

Added a renderer-neutral Arc visual asset descriptor so Arc expression decisions can represent PNG today and future Rive/GLB assets later. Existing ArcWidget PNG rendering remains unchanged.

- apps/mobile/lib/widgets/arc/arc_visual_asset.dart

## Implementation Notes

- Added `ArcVisualAsset` and `ArcVisualAssetType` for `png`, `rive`, and `glb`.
- Added `ArcAssetPaths.assetForEmotion`.
- Updated `ArcExpressionDecision` to expose `ArcVisualAsset` while preserving `assetPath` compatibility.
- Updated ArcWidget to render PNG descriptors and fall back safely for future non-PNG descriptor types.
- No 3D runtime dependency was added.

## Acceptance Check

- Arc expression decisions return a renderer-agnostic asset descriptor.
- Existing ArcWidget still renders current PNG assets.
- Tests prove PNG behavior remains unchanged.
- Future Rive/GLB types can be represented.
