# QST-061 Report: Questra Design System V1

## Summary

Questra now has a formal V1 design system layer for premium adventure UI styling. The system introduces named tokens for color, typography, spacing, radius, shadows, gradients, and app theme composition, then applies them to Home and Arc UI surfaces through shared components.

## Changes

- Added `AppColors` for Deep Navy, Cosmic Blue, Gold Accent, glass, and supporting colors.
- Added `AppTypography` for Questra headline, title, and body styles.
- Added `AppSpacing`, `AppRadius`, `AppShadows`, and `AppGradients`.
- Added `AppTheme` and `QuestraThemeTokens`.
- Updated `QuestraTheme` to delegate to `AppTheme`.
- Updated `QuestraColors` to remain compatible while using the formal color tokens.
- Updated `QuestraCard` to render glass-card styling with premium shadow treatment.
- Updated `ArcSpeechBubble` to use glass gradient, gold border, and glow shadow.
- Applied Deep Navy background to Home and Arc Chat surfaces.
- Added design system tests for theme and token availability.

## Acceptance

- Common theme is available through `AppTheme`.
- Existing `QuestraTheme.light` remains usable.
- Home uses refreshed Deep Navy and glass-card treatment.
- Arc UI uses refreshed speech bubble and glass-card treatment.
- Design direction is closer to the Questra mock direction: premium, adventurous, Deep Navy, Cosmic Blue, and Gold Accent.
