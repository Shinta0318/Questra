# QST-050 Report: Arc Celebration System

## Summary

Arc now has a centralized celebration layer for meaningful progress moments. Mission completion, Trail reflection, Quest progress, Mission creation, and Trail recording can use the same Arc-native celebration copy and expression behavior without blocking the core flow.

## Changes

- Added `ArcCelebrationService` for event-specific celebration copy and expression decisions.
- Added `ArcCelebrationCard` for inline progress celebration moments.
- Added `showArcCelebrationSnackBar` for non-blocking celebration feedback.
- Replaced Mission completion SnackBars on Home and Mission screens with Arc celebration feedback.
- Added Arc celebration feedback when a Trail Reflection is saved.
- Added an inline Quest progress celebration card for active Quests at 85% or higher progress.
- Added Arc celebration feedback for Mission creation and Trail recording in Quest detail.
- Added unit tests for Mission completion, Trail reflection, and Quest progress celebration decisions.

## Acceptance

- Celebration expression is shown for key progress events.
- Celebration copy is specific to Arc and the current journey event.
- Celebration feedback uses non-blocking SnackBars or inline cards.
- Arc remains framed as a navigator and companion.
